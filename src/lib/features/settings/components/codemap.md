# src/lib/features/settings/components/

## Responsibility
Settings UI Components — individual settings cards for the settings panel.

## Components
- `SettingsHeader.svelte` — settings panel header with title and version info
- `SettingsAiCard.svelte` — AI API URL, API key, and temperature configuration
- `SettingsTemplateCard.svelte` — Lua script template and AI script directory editor
- `SettingsShortcutsCard.svelte` — keyboard shortcut display and customization
- `SettingsPackageCard.svelte` — ZIP/YPK package include patterns configuration
- `SettingsCoverAndLog.svelte` — cover image upload and error log viewer
