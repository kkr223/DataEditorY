# src/lib/features/card-image/components/

## Responsibility
Card Image UI Components — rendering canvas, export controls, and field editing for the card image maker.

## Components
- `CardImageCanvas.svelte` — leafer-based canvas that renders the card image using `yugioh-card-ts`
- `CardImageControls.svelte` — export format (PNG/JPG), scale percentage, export/copy buttons
- `CardImageFieldEditor.svelte` — inline field editing overlay for direct text editing on the card image
